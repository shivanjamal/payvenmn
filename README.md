## ✨ تایبەتمەندیێن ئەپی (Features)

* **دیزاینەکێ مۆدێرن:** بکارئینانا ستایلێ Dark Mode و Glassmorphism.
* **سیستەمێ ئەکاونت:** دروستکرنا حسابێ و پاراستنا پەیڤان.
* **ئاسایش:** پشتگیرییا **FaceID / Fingerprint** دکەت بۆ لۆگینێ.
* **بێ وایەر:** ب ڕێکا **Expo Go** دشێی ب سانەهی تاقی بکەی بێی پێدڤی ب کێبڵێ بیت.

## 🚀 چەوان ئەپی تاقی دکەی؟ (Installation Guide)

بۆ هندێ تو بشێی ئەپێ مە تاقی بکەی، پێدڤییە ڤان هەنگاڤان جێبەجێ بکەی:

### ١. دابەزاندنا Expo Go
بەری هەر تشتەکی، پێدڤییە ئەپێ **Expo Go** ل سەر مۆبایلا خۆ دابەزینی:
- [داگرتن بۆ iOS (iPhone)](https://apps.apple.com/app/expo-go/id982107779)
- [داگرتن بۆ Android](https://play.google.com/store/apps/details?id=host.exp.exponent)

### ٢. سکانکرنا QR Code
پشتی ئەپ دابەزاند، کامێرا مۆبایلا خۆ ئاراستەی ڤی کۆدی بکە:

## 🛠 تەکنەلۆژیایێن هاتینە بکارئینان (Tech Stack)
